
Programas essenciais para instalar após formatar o Windows
==========================================================



# ---------------------------------------------------------------------------------------

* Suíte de escritório gratuita


LibreOffice é uma suíte de escritório totalmente gratuita e de código aberto que funciona 
como alternativa ao Pacote Office, que é pago e inclui programas como Microsoft Word, 
Microsoft Excel e Power Point. 


LibreOffice     => https://pt-br.libreoffice.org/baixe-ja/libreoffice-novo/

ou

WPS Office Free => https://www.wps.com/pt-BR/office/windows/

# ---------------------------------------------------------------------------------------

* Navegadores web


Firefox       => https://www.mozilla.org/pt-BR/firefox/all/#product-desktop-release

ou

Google Chrome => https://support.google.com/chrome/answer/95346?hl=pt-BR&co=GENIE.Platform%3DDesktop

# ---------------------------------------------------------------------------------------

* Reprodutor de vídeos

VLC                  => https://www.videolan.org/index.pt-BR.html

ou

Media Player Classic => https://mpc-hc.org/

MPC-HC não está em desenvolvimento desde 2017. Mude para outra coisa.

# ---------------------------------------------------------------------------------------

* Descompactador de arquivos
 
O software mais conhecido para isso é o WinRAR, que apesar de ter uma versão de 
teste vitalícia apresenta alguns anúncios em sua interface. Como alternativa temos 
o 7-Zip, que realiza as mesmas funções porém é totalmente gratuito.
 
7-Zip         => https://www.7-zip.org/


# ---------------------------------------------------------------------------------------

* Visualização de documentos no formato PDF

Master PDF Editor => https://code-industry.net/get-masterpdfeditor/

# ---------------------------------------------------------------------------------------

* Para fazer backup e sincronizar os seus documentos na nuvem.

Dropbox => https://www.dropbox.com/pt_BR/


# ---------------------------------------------------------------------------------------

* Thunderbird (Alternativa ao Microsoft Outlook) => https://www.thunderbird.net/pt-BR/

O Thunderbird é um cliente de e-mail grátis, de código aberto e multiplataforma, que 
permite que você acompanhe toda a sua correspondência de e-mail.


# ---------------------------------------------------------------------------------------

* Java => https://www.java.com/pt-BR/download/manual.jsp

CDBurnerXP   => https://cdburnerxp.se/en/download
