Programar o Windows para desligar sozinho diariamente
=====================================================



Criando uma rotina com o Agendador de Tarefas

Para quem deseja que o Windows mantenha uma rotina de desligamento periódica, é possível fazê-lo criando uma 
rotina, através da ferramenta Agendador de Tarefas:

    Abra o Agendador de Tarefas
    
    Pressione "Win+R" para abrir o Executar, digite taskschd.msc e clique em "OK" para abrir o Agendador de Tarefas. 
   
   
    
Caso deseje cancelar o desligamento, digite shutdown -a no Prompt de Comando e tecle Enter; o Windows exibirá 
uma nova mensagem, avisando sobre o cancelamento.

    
Arquivo:    desligar.bat
